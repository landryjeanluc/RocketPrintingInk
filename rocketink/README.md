# RocketInk WordPress Theme

**Version:** 1.0.0  
**Author:** Rainbow Printing  
**License:** Proprietary

## Overview
A professional, custom WordPress theme for printing services businesses. Features CMYK-inspired color scheme matching the Rocket Printing logo, fully responsive design, and optimized for secure document printing companies.

## Installation

1. Upload the `rocketink` folder to `/wp-content/themes/`
2. Activate the theme in WordPress Admin → Appearance → Themes
3. Customize contact info in Appearance → Customize → Contact Information
4. Add logos to `images/logos/` folder
5. Add equipment photos to `images/equipment/` folder

## Required Images

**Logo:** `images/logos/rocket-logo.png` (or set via Customizer)

**Chamber Logos (must use exact filenames):**
- `images/logos/hampton-chamber.png`
- `images/logos/fredericton-chamber.png`
- `images/logos/moncton-chamber.png`
- `images/logos/saintjohn-chamber.png`
- `images/logos/sussex-chamber.png`

**Equipment Photos (any filenames work):**
- Just drop JPG, PNG, GIF, or WEBP files into `images/equipment/`
- Theme automatically displays all images found
- No specific naming required - use any descriptive names you want
- Examples: `offset-press.jpg`, `color-machine.png`, `printer-1.jpg`, etc.
