# Creating Theme Screenshot for WordPress Admin

To make your theme look professional in the WordPress admin theme selector, you need a `screenshot.png` file.

## Requirements
- **Dimensions**: 1200px × 900px
- **Format**: PNG
- **Filename**: `screenshot.png` (must be exactly this name)
- **Location**: Root of theme folder (`wp-content/themes/rocketink/`)

## Option 1: Using the Preview HTML (Quick)

1. Open the file: `screenshot-preview.html` in your browser
2. Set your browser window to exactly 1200x900px or use browser dev tools
3. Take a screenshot of the entire preview
4. Save it as `screenshot.png` in the theme root folder

## Option 2: Screenshot Your Live Site (Recommended)

1. Go to your live site: `http://rainbowink.local/` or your domain
2. Set browser window to 1200px wide
3. Take a screenshot of the hero section
4. Crop/resize to exactly 1200x900px
5. Save as `screenshot.png` in `wp-content/themes/rocketink/`

## Using Browser Tools to Capture

### Chrome/Edge:
1. Press F12 to open DevTools
2. Click the device toolbar icon (or Ctrl+Shift+M)
3. Set dimensions to 1200x900
4. Press Ctrl+Shift+P and type "screenshot"
5. Select "Capture screenshot"

### Firefox:
1. Press F12 to open DevTools
2. Click the Responsive Design Mode icon (or Ctrl+Shift+M)
3. Set dimensions to 1200x900
4. Click the camera icon to take screenshot

## Quick Screenshot Using Online Tools

If you prefer, you can:
1. Open `screenshot-preview.html` in browser
2. Use a tool like:
   - Windows: Snipping Tool or Snip & Sketch (Win+Shift+S)
   - Mac: Screenshot (Cmd+Shift+4)
   - Browser extensions: "Full Page Screen Capture"

## After Creating Screenshot

1. Place `screenshot.png` in: `wp-content/themes/rocketink/`
2. Refresh your WordPress admin
3. Go to Appearance > Themes
4. Your theme should now show the preview image!

## What the Admin Will Show

Once the screenshot is in place, the WordPress admin will display:
- **Theme Name**: RocketInk Landing
- **Description**: Modern landing page theme for RocketInk Printing featuring vibrant CMYK-inspired gradients, dual hero templates, and comprehensive service showcase. Perfect for commercial printing businesses.
- **Version**: 1.0.0
- **Author**: RocketInk Team
- **Screenshot**: Your captured image

---

**Note**: The screenshot.png file is purely for the WordPress admin preview. It doesn't affect your actual website appearance.
